# 1.0.1

- Added `WaitForData(player, timeout?)` to the server API for awaiting and retrieving a player's initialized data snapshot.
- Added `WaitForData(timeout?)` to the client API for awaiting initial snapshot readiness and retrieving replicated data.
- Improved factory option type checking and autocomplete by inferring the data type directly from the supplied template.
